ABOUT

SUMMARY

I created a replica of the classic puzzle video game Tetris, where blocks fall from the sky, and players try to line them up to clear lines before running out of space

INSTALLATION & USAGE

Github Repo

INSPIRATION

I enjoyed watching the Tetris World Championship and started playing the game and loved the feeling of getting a Tetris. I wanted to see if I could replicate the game

TECHNOLOGIES

The project was developed using Python 3.74 and EasyGUI 0.96. I utilized various programming concepts such as object-oriented programming, game mechanics, and graphic rendering to create a functional and visually appealing game

FUTURE IDEAS AND WHATS NEXT

I plan to improve the game by adding the hold piece function, displaying levels as the game progresses, allowing players to pause the game, and adjusting background music
