PROJECT

TETRIS


ABOUT

SUMMARY

A replica of the classic puzzle video game of Tetris where blocks fall out of the sky and players tries to line them up to clear lines before they run out of space

INSTALLATION & USAGE

Github Repo

INSPIRATION

I enjoyed watching the Tetris World Championship and started playing the game and loved the feeling of getting a Tetris. I wanted to see if I could replicate the game

TECHNOLOGIES

This project was made with Python 3.74 and used EasyGUI 0.96

FUTURE IDEAS AND WHATS NEXT

In the future I would like to add the hold piece function, show the levels as the game progress, pause game and adjust background music
