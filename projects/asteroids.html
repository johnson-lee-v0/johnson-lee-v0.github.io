ABOUT

SUMMARY

A replica of the 1979 space themed video game made by Atari. The player's objective is to control the ship and dodge/destroy the incoming asteroids to prevent a crash

INSTALLATION & USAGE

Github Repo

INSPIRATION

I came across a video of someone playing the game and wanted to see how difficult it would be to code it

TECHNOLOGIES

This project was made with Python 3.74 and used EasyGUI 0.96

FUTURE IDEAS AND WHATS NEXT

In the future I would look to add a time limit to each level so it increases the difficulty and also limit the amount of ammunition that can be fired.
