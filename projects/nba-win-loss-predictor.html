ABOUT

SUMMARY

This project aimed to investigate the relationship between Regular Season Wins and Home Team Statistics over the course of the game

INSTALLATION & USAGE

Github Repo

INSPIRATION

When I was watching a NBA Game and by halftime the scores was still close between the two team. I wanted to see if I could predict the game outcome using the statistic from the first half. This lead to an article I read a while back from ESPN about how the home team wins 60% of the time, which swayed the question to can I create a model that predicts if the home team wins

TECHNOLOGIES

This project was made with Python 3.74

FUTURE IDEAS AND WHATS NEXT

In the future I would look to add more training data as the current model is trained with the 2021-2022 Regular Season data and enhance the model as the current version is just a model that can take inputs.
