ABOUT

SUMMARY

I investigated the relationship between Regular Season Wins and Home Team Statistics over the course of NBA games. The aim of this project was to create a model that predicts if the home team wins

INSTALLATION & USAGE

Github Repo

INSPIRATION

My project was inspired by an article I read from ESPN about how the home team wins 60% of the time. To test this hypothesis, I used Python 3.74 and employed statistical techniques and machine learning algorithms to analyze Regular Season Wins and Home Team Statistics data

TECHNOLOGIES

The data extraction was done through using BS4 and scrapping Basketball Referenece site. The data cleaning and creatiion of new variables was completed through the assistance of Pandas and Numpy Library. In terms of modelling LTSM and Random Forest Model were created in attempt to predict if the home team would wins

FUTURE IDEAS AND WHATS NEXT

I plan to enhance the current model by adding more training data beyond the 2021-2022 Regular Season data. Additionally, I will explore more advanced machine learning techniques to improve the model's performance and accuracy
