ABOUT

SUMMARY

I created a replica of a classic video game in which alien ships move towards the bottom of the screen, and the player's objective is to destroy them to defend Earth. The player must also shoot bullets and dodge asteroids to survive

INSTALLATION & USAGE

Github Repo

INSPIRATION

I played the classic video game on the airplane and wanted to see if I was able to replicate it and how the mechnaics of the game worked

TECHNOLOGIES

The project was developed using Java, and I utilized various programming concepts such as object-oriented programming, game mechanics, and graphic rendering. These concepts were critical in developing a functional and visually appealing game

FUTURE IDEAS AND WHATS NEXT

I plan to improve the game by fixing the background image, adding different enemy ships, power-ups, and more. These enhancements will improve the game's functionality
