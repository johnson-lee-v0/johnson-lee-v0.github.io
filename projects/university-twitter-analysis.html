PROJECT

UNIVERSITY TWITTER ACCOUNT ANALYSIS


ABOUT

SUMMARY

A further extension of a project done for my social analytics course. Using 3200 tweets each from University of Western, University of Waterloo and University of Toronto during the same time span. The project aimed to identify how the University of Waterloo could better attract prospect with Twitter. The most frequent terms in twwets, tweets sentiments and entities involved were analyzed

INSTALLATION & USAGE

Github Repo

INSPIRATION

This project was done as a further extension to apply what I learnt in my social analytics course

TECHNOLOGIES

This analysis in the project was conducted with R. The package used includes: openNLP, dplyr, tm, ggplot2, reshape2, syuzhet and more

FUTURE IDEAS AND WHATS NEXT

This project focused on examining the tweets from the universities account which only included RT or direct posts. In the future I would like to conduct a similar analysis on a trending topic where there are many different users and a connection web can then be plotted
